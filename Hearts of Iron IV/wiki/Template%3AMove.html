Move can be used to highlight articles with a request for a name change. The template calls for discussion.

Parameters

Parameter Description Default Status
1 Proposed new pagename empty required

Usage

This table may be added by copy-pasting the template as shown below into an article. 

{{Move| <--insert here the proposed new pagename--> }}

Updating this template

You shouldn't have to.

Example

{{Move|Template:Move}}

Will display this:

It has been proposed below that Template:Move be renamed and moved to Template:Move.
The discussion to support or oppose the move should be on this talk page, usually under the heading “Requested move”. Participate please.
If, after a few days, a clear consensus for the page move is reached, please move the article and remove this notice, or request further assistance.
