Variables[edit]

commander= The commander of the unit. e.g commander=LTC HERMAN ANDERSEN - default is nothing
width= The width of the space that the entire object has. e.g width=180px - default is 200px
name= Name of the military unit e.g name=1st Infantry Division - default is nothing
location= Where the unit is based e.g location=NORTHERN IRELAND - default is nothing
size= The size of the unit. size =
  • armygroup (use for complete armed forces),
  • army
  • corps
  • division
  • brigade
  • regiment
  • battalion
  • company
  • platoon
  • squad - default is nothing
type= The type of unit type=
  • infantry
  • artillery
  • antiaircraft
  • armoured
  • combinedarms
  • helicopter
  • fixedwing
  • ship
  • specialforces
  • specialoperations
  • recon
  • cavalry
  • supply
  • hq
  • staff - default is blank box
special=, special2= Any specialty the unit has special=/special2=
  • mountain
  • amphibious
  • parachute
  • airmobile
  • recon
  • airborne
  • specialforces
  • specialoperations
  • staff
  • hq
  • mechanized
  • armoured
  • motorized
  • wheeled
  • navy - default is nothing
